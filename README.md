## 二、项目需求

### 1. 软件名称：图片空间

### 2. 主要功能：图片管理软件，能管理和分享本机的图片
- 多种方式（列表、缩略图）形式显示图片。
- 展示大图，支持缩放，并支持简单的编辑（旋转，缩放等）。
- 支持上传图片到服务器。
- 使用本地 SQLite 数据库存储图片的额外信息，并支持增、删、改、查。
- 使用缩略图缓存，加速列表中图片显示。
- 类似于朋友圈的分享图片的功能。
### 3. 开发环境和工具：
- IDE：Android Studio 4.1.1，IntelliJ IDEA 2020.3
- JAVA：jdk-1.8.0_241_windows-x64
- 数据库：MySQL8.0，SQLite 3.34.0
